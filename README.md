# Ralph Demo

Using only what's in .scripts/ and .seed/ generated this repo which is an Excel clone in the browser.

# Prerequisites

- Node installed (to run the pretty-print script)
- Cursor agents cli installed and logged-in

## Steps to reproduce

1. Chuck .seeds/1-plan.md into cursor to generate the TODO.md
2. Run ./.scripts/ralph.sh to run the Ralph loop
3. Let it run for as long as you want
